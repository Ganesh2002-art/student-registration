# 🎓 Student Registration System (React)

This project is a student registration system built using **React** only. It allows the creation and management of course types, courses, course offerings, and student registrations.

## ✅ Features

- Add/Edit/Delete **Course Types** (e.g., Individual, Group, Special)
- Add/Edit/Delete **Courses** (e.g., English, Hindi, Urdu)
- Create/Delete **Course Offerings** by combining Course + Type
- Register students for specific course offerings
- Edit/Delete student registrations
- Filter offerings based on course type
- Prevent duplicate registrations
- Data persistence using `localStorage`
- Fully responsive UI (mobile/tablet/desktop)
- Built with functional components and React Hooks
